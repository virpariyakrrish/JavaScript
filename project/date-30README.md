# 📌 Project Overview


🖼️ **Image Slider** is a beautiful and responsive web application created using **HTML, CSS and JavaScript**.

This project provides a smooth experience for displaying multiple images with stylish navigation controls.

✨ The main goal of this project is to create a simple, fast and attractive image carousel with a modern UI design.


<br>

# ⚡ JavaScript Functionality


## 📸 Image Array


```javascript

var images = [

 "img1.jpg",
 "img2.jpg",
 "img3.jpg",
 "img4.jpg",
 "img5.jpg"

];

```


Images are stored inside an array and displayed dynamically.


<br>


## ➡️ Next Image Logic


```javascript

function showNext(){

 index++;

 if(index >= images.length){

 index = 0;

 }

 showImage();

}

```


<br>


## ⬅️ Previous Image Logic


```javascript

function showPrev(){

 index--;

 if(index < 0){

 index = images.length - 1;

 }

 showImage();

}

```
